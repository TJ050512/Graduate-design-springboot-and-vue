## 📝 添加新测试账号

如果需要添加新的测试账号，可以通过以下方式：

### 方式1：在前端添加
1. 使用管理员账号登录
2. 进入"用户管理"
3. 点击"添加用户"
4. 填写用户信息并保存

### 方式2：在数据库中直接添加

```sql
-- 添加普通用户（密码：123456）
INSERT INTO `sys_user` (`username`, `password`, `real_name`, `gender`, `phone`, `user_type`, `status`) 
VALUES ('user002', 'e10adc3949ba59abbe56e057f20f883e', '王五', 1, '13800138003', 2, 1);

-- 添加抄表员（密码：123456）
INSERT INTO `sys_user` (`username`, `password`, `real_name`, `gender`, `phone`, `user_type`, `status`) 
VALUES ('reader002', 'e10adc3949ba59abbe56e057f20f883e', '赵六', 1, '13800138004', 3, 1);
```

## ⚠️ 注意事项

1. **默认密码安全性**：生产环境请及时修改所有默认密码
2. **账号状态**：如果账号被禁用（status=0），将无法登录
3. **密码加密**：系统使用MD5加密存储密码，如需修改密码，请使用系统提供的修改密码功能
